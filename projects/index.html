Projects

This is a list of selected projects that I've completed over the past 2 years. They span the languages Java, C, C++, Python, x86 Assembly, and MATLAB. These projects consist of low-level systems programming with registers and messing with threads all the way to high-level image manipulation and developing games.

Random Text Generator

Generates random text similar to a given input. Uses the Markov property as illustrated in the Markov Chain to predict the next character in the output. At a low level, the output should have a similar proportion of characters as the input, but as the level increases, it should start to resemble the language and diction of the input.

Tetris

Implemented the game of Tetris in Java with a UI that displayed the score. Included a Brain that gave hints and could even play the game on its own.

Boggle

Designed the game of Boggle in Java and included a simple text-based UI.

WebCrawler

Developed a highly efficient search engine in Java that crawls the Web. Supports various types of queries: single word, AND, OR, negative, phrase, implicit AND. Composed of a recursively managed Query Parser for quick lookup time.

Matrix Balance Cache

Came up with an efficient way to check whether a matrix was balanced or not. Involved taking advantage of the properties of the cache.

FUN Features

Added features for a basic language called FUN that supports expressions, statements, and variables. Implemented a linkage convention to support functions and variable scopes as well as convert FUN to Assembly. Executed a parsing technique that allows one to define and implement classes as well as instances of that class.

Preemptive Threads

Added the concept of threads and processes to a bare Operating System. Included the functionality for preemptive threads so that more important tasks can be completed first.

BobFS

Constructed an ext2-like file system with inodes and blocks. Allows directories and various file types.

Virtual Memory

Provided in-kernel virtual memory support and handled page faults elegantly. Accounted for shared, identity, zero-filled, and private mapping.

TearsOS

Assembled a rudimentary, but fully functional operating system as part of a team. Consisted of a heap, threads, processes, virtual memory, and a file system in kernel and user mode. Added a Copy-On-Write feature to optimize the management of virtual memory.

Content-Aware Image Resizing

Resizes images in a manner that preserves important content. Uses dynamic programming to remove or add the best seam.

Texture-Based Segmentation

Segments input image into k sections based on texture. Provides a different view on classifying parts of an image.

Circle Detection

Detects circles of a given radius in the input image.

Image Mosaics

Creates a mosaic of 2 or more images from different angles of the same scene.

Visual Search & Rank

Given an input image, searches for and ranks from images in a database that are from the same scene.

Face Detector

Implemented a Dalal and Triggs multi-scale sliding window detector to recognize human faces. Trained a linear classifier and added hard negative mining to improve accuracy. Achieved an average precision of 93%.

Alien Invasion

Created an application to play a simple game of alien invasion in Python.
